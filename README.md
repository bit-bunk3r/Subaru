# Subaru
Bot oficial do BitBunker!

## Dependencias
- Go 1.20
- Um token do Discord válido.

## Executando
1. Copie o `.env.example` para `.env`
2. Configure o `.env` com as informações adequadas.
3. Rode no terminal: `go run cli/subaru/main.go`
